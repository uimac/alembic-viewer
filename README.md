# alembic
alembic (.abc) viewer

## Install
```
npm install alembic-viewer
```

## Start
```
npm start
```

## Features

#### Format
 - .obj
 - .abc

#### abc support primitives

|primitive|current status|
| --------------- |---------------|
| mesh | o |
| point | o |
| curve | x |
| nurbs | x |
| camera | x |
| material | x |
| light | x |
